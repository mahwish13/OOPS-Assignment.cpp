Modules Overview

Module I – Basics (1–40)
Topics: Syntax, variables, operators, loops, conditionals, arrays, strings, pointers, structures
Examples: Hello World, Swap Numbers, Leap Year, Matrix Addition

Module II – Classes & Objects (41–80)
Topics: OOP fundamentals, constructors, destructors, member functions
Examples: Bank Account, Calculator, Employee Details

Module III – Advanced OOP (81–120)
Topics: Inheritance, polymorphism, operator overloading, virtual functions, abstract classes

Module IV – C++ I/O (121–145)
Topics: File handling, streams, manipulators, stringstreams

Module V – Exception Handling (146–170)
Topics: try-catch, multiple exceptions, user-defined exceptions, safe file operations
